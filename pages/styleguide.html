Nav Link Styles
  • Font size: 0.88rem, weight 500
  • Color: --gray-600 (default), --black (hover/active)
  • Active indicator: 18px blue underline bar, 2.5px thick
  • CTA button: .btn .btn-blue .btn-sm in nav-links
Dropdown Menu (.dropdown-menu)
  • Position: Absolute, centered below parent
  • Background: White with --shadow-lg
  • Border radius: var(--radius) = 16px
  • Item hover: --blue-50 background, --blue text
  • Animation: Fade in + translateY from 8px to 0

Mobile Menu (1024px and below)

  • Trigger: Hamburger button .nav-toggle (3 spans animating to X)
  • Menu: Full-width dropdown below navbar
  • Links: Stacked vertically, 16px 28px padding
  • Dropdown: 2x2 grid of bordered cards
  • CTA: Button at bottom with 16px 28px margin
  • Max height: 85vh with overflow-y: auto
